# rpi-buttons
Scripts to manage the Raspberry Pi (GPIO buttons)

## Files:

- rpibtn.py: script to manage the six-buttons rpi case
- rpibtn_ir.py: script to manage the buttons in the Rapsberry Pi Infrared Shield
